# About this Project
The purpose of this project is to build a website for the Pace Coffee farm in Nago, Okinawa.

# License
This project is unlicensed.

# Getting Started
## Frontend
```
cd frontend
yarn install
yarn dev
```

## Backend
```
docker run --name pace-db -e POSTGRES_PASSWORD=root -d postgres
cd backend/
yarn install
yarn dev
```
